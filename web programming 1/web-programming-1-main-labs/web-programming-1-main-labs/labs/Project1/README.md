# Rosemary Pine

This is a shopping website for the clothing brand, Rosemary Pine&trade;

## Installation

Install like any other website

```
$ git clone https://git.cs.du.edu/osccerin/web-programming-1.git
$ cd ../path/to/the/file
$ npm install
$ npm start
```
